# landing-page-psicologo
Desarrollo de una landing page personalizada con HTML5, CSS3 y JavaScript (ES6+).

# Demo
https://abomdevp.github.io/landing-page-psicologo/

# Funcionalidades
- Estructura de negocio
- Navegación responsive
- Validación básica de formularios
- Animaciones CSS sutiles
- Accesibilidad WCAG 2.1 AA

# Uso
1. Clonar o descargar el proyecto
2. Abrir index.html en el navegador
3. Listo (no requiere instalar dependencias)
